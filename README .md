# ShopEase - E-Commerce Product Catalog

## Final Internship Capstone Project

ShopEase is a responsive E-Commerce Product Catalog web application built using HTML, CSS, and JavaScript.

## Features

- Responsive multi-page website
- Product catalog
- Product search
- Category filter
- Add to cart
- Remove from cart
- Cart total calculation
- Local Storage support
- Contact page
- About page
- Clean and professional UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- DOM Manipulation
- Local Storage
