About ShopEase

Final Internship Capstone Project

Project Overview

ShopEase is a responsive E-Commerce Product Catalog developed using HTML, CSS, and JavaScript as part of a Web Development Internship Final Project.

This application demonstrates modern frontend development, responsive layouts, dynamic product rendering, shopping cart functionality, product search, category filtering, and local storage integration.

Project Features

  • Responsive Design
  • Multiple Pages
  • Dynamic Product Catalog
  • Search Products
  • Category Filter
  • Add to Cart
  • Shopping Cart Management
  • Local Storage Support
  • Professional UI Design

Technologies Used

HTML5 CSS3 JavaScript (ES6) DOM Manipulation Local Storage Responsive Web Design
